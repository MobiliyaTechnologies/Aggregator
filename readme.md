# Aggregator Server

## Pre-requisites 

- NodeJS (4.0.0 and above)
- Python (2.7 and above)
- OpenCV 3+

## Installation

```bash 
$ git clone 
$ cd jestson-device-client
$ npm install
```

## Running the server

Starting the aggregator server:

``` bash
$ node jetsonServer.js

```
